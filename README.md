# Development of a Hybrid Encryption System

## Crypto_App

A desktop application that implements **hybrid encryption** combining AES-128 (EAX mode) for fast data encryption and RSA-2048 (OAEP padding) for secure key exchange.

## Features

- 🔐 Encrypt / Decrypt text messages
- 🖼️ Encrypt / Decrypt images (with preview)
- 📁 Encrypt / Decrypt files
- 🎥 Encrypt / Decrypt videos (with progress tracking)
- 🔑 RSA key management (generate, view, export in PEM format)
- ⏱️ Real-time progress bars and elapsed time display

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python 3.13 | Main programming language |
| PyCryptodome | AES + RSA encryption |
| CustomTkinter | Modern graphical user interface |
| Pillow (PIL) | Image processing and preview |
| NumPy | Numerical operations |
